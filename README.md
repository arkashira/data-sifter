# Data Sifter
A Python project for web scraping and data extraction.

## Features
* Extracts data from a single website
* Supports multiple websites
* Handles different data formats

## Usage
1. Create a `DataSifter` object
2. Add websites using the `add_website` method
3. Extract data using the `extract_data` method
4. Get websites using the `get_websites` method
5. Get a website using the `get_website` method

## Testing
Run tests using `pytest` command.
